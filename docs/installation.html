Installation

Prepare the environment before starting the resource.

  1. Confirm VORP Core and oxmysql are installed and functioning.
  2. Import the SQL file supplied with the purchased package.
  3. Place the resource in the server resources directory.
  4. Review the configuration before first start.
  5. Ensure dependencies start before Hypas Law MDT.
  6. Restart the server and check console output for database or configuration errors.

The purchased release documentation remains authoritative for exact resource names and version-specific steps.
